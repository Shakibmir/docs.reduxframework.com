    Checks dependencies between objects based on the $field['required'] array

    check_dependencies(array $field) : array
    If the array is set it needs to have exactly 3 entries. The first entry describes which field should be monitored by the current field. eg: "content" The second entry describes the comparison parameter. eg: "equals, not, is_larger, is_smaller ,contains" The third entry describes the value that we are comparing against. Example: if the required array is set to array('content','equals','Hello World'); then the current field will only be displayed if the field with id "content" has exactly the value "Hello World"
